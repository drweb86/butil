Usage

Configurator 'argument'

Command Line Arguments:

RemoveLocalSettings

Removes configuration data of current user. It can be used for resetting settings. Also this removes default logs folder(the logs folder that is set up at the very first program start)

JustRestorationMaster

Runs only restoration Master

JustBackupMaster

Runs only Backup Master

file.BUtil

Shows restoration Master with this file opened
